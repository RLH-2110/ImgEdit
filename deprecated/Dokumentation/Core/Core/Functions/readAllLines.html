Overview

This function loads a file as a string array, but only if the file is not too big.

Definition

public static string[] readAllLines(string path, ulong filesizeLimit,bool console)

The function takes a string, a long and a bool as parameters.
The string is the file path to the preset file from which we want to read the MediaFormat from.
The long is the file size limit of the file we want to read. This is done, so that the user may not load a 50 Petabyte configuration file into memory.
The bool is to indicate if it is run by the console version. This is only used for the Output (Core.cs - Core) function.

The return is a string array, that contains all the lines we have read

Behavior

readAllLines checks if the file exists, and if the lenght is biger than the file size limit, if either is the case, we output an error and return NULL.
If the cheks are passed, we just return the normal read all lines function: return System.IO.File.ReadAllLines(path);
